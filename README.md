# Media Indexer

Media indexer provides a convienent way to index, search and play media files.

Supported Indexing Properties:
1. Artists
2. Categories
3. Tags

 Supported Media Types:
1. Audio (.mp3)
2. Video (.mp4, .avi)

With option to add other file extentions
